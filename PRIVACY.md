# Privacy Policy for NeTab

Last updated: June 18, 2026

NeTab is a local-first Chrome New Tab extension.

NeTab does not sell, share, transfer, or rent user data. The extension does not operate a remote backend service and does not require an account.

NeTab stores dashboard configuration locally in the browser. This may include custom quick links, categories, selected icons, media preferences, display settings, and media uploaded by the user through the extension settings panel.

NeTab uses Chrome local storage and browser storage APIs such as IndexedDB to keep this information on the user's device. Users can edit, export, reset, or remove their configuration from the extension settings UI.

NeTab does not collect browsing history for analytics or advertising. NeTab does not collect financial information, authentication information, health information, personal communications, or precise location information.

The extension may use Chrome's favicon capability to display icons for user-configured links. This is used only to make the dashboard easier to read.

If the public source code is customized by another developer, that developer is responsible for updating this privacy policy to match their modified version.
