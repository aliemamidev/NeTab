# Chrome Web Store Listing Draft

## Title

NeTab — Custom New Tab Dashboard

## Summary

A clean, customizable New Tab dashboard with quick links, widgets, wallpapers, videos, and local-first settings.

## Category

Productivity

## Detailed description

NeTab replaces your default Chrome New Tab page with a beautiful, customizable personal dashboard.

Organize your favorite websites into categories, manage quick links, choose icons, enable or disable sections, and personalize your browser start page with image or video backgrounds. Everything is designed to feel clean, fast, and simple.

Main features:

- Custom quick links and categories
- Built-in settings panel
- Local image and video background support
- Local media management
- Lightweight built-in widgets
- Sample links and media included
- Local-first configuration
- No account required
- No tracking-focused experience

NeTab is useful for developers, students, creators, gamers, and anyone who wants a cleaner and more personal browser start page.

Your personal links and media can stay local, while the public version of the project includes safe example content that users can customize after installation.

## Permission justification

### storage

Used to save dashboard preferences, custom links, categories, media settings, and UI configuration locally in the browser.

### favicon

Used to show favicons for user-configured quick links. Favicons are used only for visual identification of links.

## Data usage statement

NeTab stores user configuration locally in the browser. It does not sell user data, does not use user data for advertising, and does not transfer user data to a remote backend.
